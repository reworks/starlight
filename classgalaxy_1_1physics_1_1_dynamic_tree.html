Obtains collision candidates for the AABB associated with the specified ID.

In order to avoid unnecessary dynamic allocations, this function returns the resulting collision candidates through an output iterator. This means that it is possible to write collision candidates to both a stack buffer and something like a std::vector.

The output iterator can for instance be obtained using std::back_inserter, if you're writing to a standard container.

This function has no effect if the supplied key is unknown.

Parameters
keyThe ID associated with the AABB to obtain collision candidates for.
[out]iteratorThe output iterator used to write the collision candidate IDs.
